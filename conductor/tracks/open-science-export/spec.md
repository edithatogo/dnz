# Track Specification: Open Science Export

## Overview
This track provides tools to export harvested data to open research formats. It generates Data Package specifications (Frictionless Data) and automates metadata creation for uploads to Zenodo and OSF.

## User Stories / Requirements
- As a researcher, I want to export harvested search results with complete metadata schemas in Frictionless Data Package format.
- As a publisher, I want automated API uploads of datasets to public repositories (like Zenodo and OSF).
- As a user, I want JSON-LD schemas automatically compiled for dataset reproducibility.

## Technical Constraints
- Follow the Frictionless Data Package standard.
- Use Zenodo and OSF REST APIs.
