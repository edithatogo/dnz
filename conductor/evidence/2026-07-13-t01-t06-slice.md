# T01/T02/T06 implementation slice evidence

## Implemented

### Review blocker fixes in current slice

- Added public `DnzError` variants for HTTP status, decode, and transport failures without embedding URLs or credentials.
- Bounded default HTTP timeout to 30 seconds and added `with_timeout` override.
- Parse numeric `Retry-After` values with a 60-second cap and use them for retry delays.
- Removed raw transport error logging, added HTTPS validation with loopback-only HTTP test support, and separated header versus legacy-query cache namespaces.
- Updated integration fixtures to assert header authentication by default.
- Removed the dense harvest partition `min(10)` page cap.
- Added validated bbox and protected-key-safe extra query parameter APIs; facet page size is capped at 350.

- `Client::unauthenticated()` provides explicit public mode while preserving `Client::new(token)`.
- Configured tokens use the `Authentication-Token` header by default.
- `with_legacy_query_key_auth()` is the only path that adds `api_key` to the query.
- Cache keys include a non-secret authentication namespace, preventing public and authenticated identities from sharing responses.
- Request debug parameters redact legacy query credentials.
- `per_page(0)` is serialized exactly as zero for count/facet-only requests.
- RAG XML escapes IDs, attributes, text, partners, descriptions, and dates.
- Record deduplication retains empty-title records and uses non-empty IDs as the primary identity before title fallback.
- Timeline ordering extracts a four-digit date key where possible and retains the original display value.
- Citation output is labelled as generic references rather than claiming MLA/APA compliance.
- `MemoryVectorStore` now enforces one embedding dimension and upserts repeated record IDs rather than retaining stale duplicates.
- Embedding model downloads now require HTTPS (with loopback-only HTTP test support), reject redirects, and retain safe single-component filenames.
- The Python adapter now reuses one lazily initialized Tokio runtime instead of constructing a runtime per request.
- Schema.org exports no longer assert a dataset-wide CC0 license or fabricated temporal coverage when source evidence is absent.
