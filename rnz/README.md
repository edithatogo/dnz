# RNZ Recordings and Associated Documents

This bundle captures the RNZ-related collection metadata and supporting documents that are already checked into the repo-local DigitalNZ exports.

## Contents

- `collections.csv`: RNZ-related collection names and counts from `digitalnz/facets/collection.csv`.
- `usage_by_collection_and_partner.csv`: Radio New Zealand partner rows from `digitalnz/facets/usage_by_collection_and_partner.csv`.
- `recordings.md`: curated inventory of the RNZ recording and document records represented in the checked-in facet export.

## Notes

- The bundle is derived from checked-in source artifacts, not live API calls.
- Archive workflows mirror this bundle alongside the existing source trees.
